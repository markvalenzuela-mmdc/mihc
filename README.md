## Folder Structure

- `nextjs/`: Contains the main web application code. The main app is a Next.js app.
- `playwright/`: Contains the playwright test suite for end-to-end testing of multiple internal apps.
- `docs/`: Contains documentation related to the project, including design documents, pivots, and other relevant information.
- `docker/`: Contains Docker Compose files and configurations for containerizing the playwright test suite and nextjs app.

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
    cd <project-directory>
   ```
